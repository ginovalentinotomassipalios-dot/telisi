# Telisi v0.2.1

Mejora visual sobre Telisi v0.2.0.

## Cómo abrir

1. Descomprimir este ZIP.
2. Abrir la carpeta `telisi_v0_2_1` en Visual Studio Code.
3. Abrir terminal.
4. Ejecutar:

```powershell
npm.cmd install
npm.cmd run dev
```

## Novedades

- Modo oscuro/claro.
- Botón de cambio de tema en la esquina superior derecha.
- Estadísticas rápidas de eventos de hoy por calendario.
- Mensajes inteligentes según la cantidad de eventos.
- Animación suave al cambiar de sección.
- Exportación de backup en Ajustes.
- Versión actualizada a Telisi v0.2.1.
