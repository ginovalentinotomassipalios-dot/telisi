# Telisi 0.12.2

- Se eliminó la tarjeta fija de creación de eventos del calendario móvil.
- Al tocar un día del mes se abre el modal de nuevo evento.
- El modal recibe automáticamente la fecha elegida y reinicia hora, título, recordatorio y recurrencia.
- Se conserva debajo del calendario la lista cronológica de todos los eventos del mes.
- Se recompiló `dist` y se sincronizó el proyecto Android con Capacitor.
